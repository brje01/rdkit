Installation Guide

There are different ways to install the RDKit, depending on your needs and environment. The easiest way is to use the conda package manager.

conda install rdkit

If you are a developer or need the latest features, you can build RDKit from source. There is a blog entry on Building RDKit from Source that provides detailed instructions.
